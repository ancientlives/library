### D3 - Drawing - SVG Axes

##### Intro
In D3, *SVG axes* are, in fact, built-in functions with definable parameters. When we call an axis function in D3, it doesn't simply return a value. Instead, it creates the defined elements for the visual rendering of an axis. This includes lines, labels, and other associated details.
